SNOWY DRIZZLE

Snowy Drizzle is a responsive and interactive ice cream shop website built using HTML, CSS, and JavaScript. It features a modern UI with multiple ice cream categories, a search filter, and a dynamic shopping cart system with add, remove, and quantity controls. The project also includes a dark/light theme toggle and a contact form. It demonstrates frontend development skills along with local storage for cart data persistence.
